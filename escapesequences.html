Escape Sequences

Escape Sequences start with a backslash ( \ )
We use them in some cases to help us with certain things that we will see later.
There are many escape sequences , but we will only cover the basic ones.

We start with \n , it moves us down to the next line.

Example :

We will get Hello Hi in the same line ,
but if we write the same code with \n :

We will get this in two different lines :
Hello
Hi

We will normally get the result Hello Hi with one space ,
but if we write the same code with \t :

We will get Hello      Hi , with 4 spaces ( tab ) .



Next , we start with \b ( backspace ) , it erases the character right before it in the output.

Example :

We will get HellHi , because the \b removes the letter o , since o was the character right before it.



Next , we start with \r , it deletes everything before it on the same line , and the output starts from whatever comes after the \r.

Example :

We will get my name is Mahmoud , because the \r deleted the word Hello.



Next , we start with \a , its not very important , but I mentioned it for reference.
It makes a beep or alert sound.
When the program reaches the line \a itself , it triggers a sound ( like a system alert ) .
